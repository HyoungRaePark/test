# TIL - 서버 로그 분석 대시보드 구현

## 오늘 배운 내용

오늘은 로그 분석 결과를 JSON으로 저장하고, HTML 대시보드에 데이터를 연결하는 작업을 진행했다.

* Python에서 집계한 데이터를 `results.json`으로 저장하는 방법을 익혔다.
* Chart.js에서 사용할 수 있도록 Python 리스트를 JSON 형태로 변환하여 HTML에 삽입하는 과정을 구현했다.
* 상태 코드, 시간대별 요청 수, 에러 URL TOP 5, 요청 수 TOP 5 IP 데이터를 각각 차트에 연결했다.
* 딕셔너리 데이터를 원하는 순서로 출력하기 위해 `sorted()`를 활용하여 상태 코드를 숫자 기준으로 정렬하는 방법을 배웠다.
* JSON의 키 이름(`top_ip`, `top_ips`)이 일치하지 않으면 `KeyError`가 발생한다는 점을 확인하며 데이터 구조를 일관성 있게 관리하는 것이 중요하다는 것을 알게 되었다.

## 느낀 점

기존에는 데이터를 출력하는 것에만 집중했는데, 오늘은 분석 결과를 시각화하면서 데이터를 보기 쉽게 전달하는 과정도 중요하다는 것을 느꼈다. 단순히 그래프를 만드는 것뿐만 아니라 레이아웃과 디자인까지 조금씩 수정해 보면서 대시보드의 완성도를 높이는 방법도 배울 수 있었다.
